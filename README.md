Accelerometer
===================

Application to Showcase Accelerometer APIs


# Purpose
This Application will showcase the following features of Accelerometer APIs

* Current Accelerometer Data
* Accelerometer Monitoring
* Register Shake Event

# Supported platforms:
**Mobile**
 * Android
 * iPhone
 * windows phone 7.5
 * windows phone 8

**Tablet** 
 * Android
 * iPad
 * Windows8
